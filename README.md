# fmri_ABIDE
基于自闭症的公开数据集(ABIDE)中fMRI数据的处理，以及分类

🎯 研究目标：复现并改进两篇前沿fMRI分析方法，提升自闭症谱系障碍(ASD)分类性能

📚 核心文献：
1. HDLFCA (MedIA 2022): An attention-based hybrid deep learning framework integrating brain connectivity and activity
   🔗 https://doi.org/10.1016/j.media.2022.102413

2. BolT (MedIA 2023): Fused window transformers for fMRI time series analysis
   🔗 https://doi.org/10.1016/j.media.2023.102841

🗂️ 数据集：ABIDE I (Autism Brain Imaging Data Exchange)
   • ASD: 743 subjects | HC: 779 subjects
   • 多中心、多扫描参数、静息态fMRI
